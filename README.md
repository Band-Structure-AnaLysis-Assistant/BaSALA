# BaSALA (Band Structure AnaLysis Assistant)

**BaSALA** is a dedicated analysis assistant software designed for researchers in solid-state physics and materials science to intuitively analyze electronic band structures.

## Supported Methods
Currently, BaSALA specializes in analyzing spectra obtained from the following techniques:

* **XPS** (X-ray Photoelectron Spectroscopy):
  - Band gap (Eg) analysis from narrow spectra.
  - Valence Band Maximum (VBM) determination from low-binding-energy (0-20 eV) region measurements.
* **UPS** (Ultraviolet Photoelectron Spectroscopy): Work Function (WF), Ionization Potential (IP), and HOMO onset determination.
* **LEIPS** (Low-Energy Inverse Photoemission Spectroscopy): LUMO onset determination.
* **LEET** (Low-Energy Electron Transmission): Vacuum Level (VL) determination via inflection point analysis.

We are constantly updating! More analytical methods and instruments will be supported in future releases.

## How to Run
Simply download the latest `.exe` file from the **Releases** page and run it on Windows. No Python environment installation is required!
